Alle wichtigen Änderungen am RTC Import System werden hier dokumentiert.

### Geplant

## [1.0.0] - 2026-04-24

### Added
- Initial Release
- Python-basiertes Import-Script mit direktem DB-Zugriff
- Automatisches Track-Mapping
- Automatische GT7-Version-Erkennung via Datum
- Team/Fahrer Auto-Discovery
- Batch-Import-Script für komplette Saison
- DB-Backup-Script
- Vollständige Projekt-Dokumentation
- Git-Setup mit .gitignore und Commit-Templates

### Features
- CSV-Import mit Auto-ID-Generierung
- Grid-Klassifizierung (1, 2a, 2b, 3)
- DNF-Erkennung
- Penalty-Parsing
- Punkte-Berechnung
- Validierung nach Import

### Documentation
- README.md mit vollständiger Anleitung
- QUICKSTART.md für schnellen Einstieg
- Inline-Kommentare im Script
- Beispiel-Environment-Datei
